# Next.js & GraphQL Rick and Morty API client

## How to use

Install it and run:

```sh
yarn install
yarn dev
```

Install it and run for production:

```sh
yarn install
yarn start
```

Run tests:

```sh
yarn test
```

**[Live demo](https://<waiting-for-deploy>.now.sh/)**.

- [Next.js](https://nextjs.org/)
- [Apollo-Client](https://www.apollographql.com/docs/react/)
